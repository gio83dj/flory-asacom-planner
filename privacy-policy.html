Cos'è Flory Planner

Flory Planner è un'applicazione Android personale progettata per assistenti scolastici e operatori educativi. Permette di registrare le ore di lavoro con gli alunni, visualizzarle su un calendario mensile, consultare un'agenda giornaliera e generare un resoconto mensile delle ore svolte.

L'app è pensata per uso individuale e non prevede account, registrazioni o servizi online.

✅ Nessun dato viene raccolto

Flory Planner non raccoglie, non trasmette e non condivide nessun dato personale con terze parti, sviluppatori o server esterni.

  • Nessuna connessione a internet richiesta
  • Nessun account o registrazione necessaria
  • Nessun dato inviato a server esterni
  • Nessuna analisi, tracking o pubblicità
  • Nessuna richiesta di permessi sensibili
💾

Come vengono salvati i dati

Tutti i dati inseriti (nomi degli alunni, scuole, sessioni, orari) vengono salvati esclusivamente sul dispositivo dell'utente, nella memoria privata dell'app tramite la tecnologia localStorage del browser interno (WebView).

I dati non lasciano mai il dispositivo, a meno che l'utente non esporti manualmente un file di backup tramite la funzione "Esporta" dell'app.

📤

Funzione di Export/Import

L'app permette di esportare i dati come file JSON e di importarli in seguito. Questo file viene creato sul dispositivo e condiviso tramite le app di condivisione di Android (es. email, WhatsApp, Google Drive) a esclusiva scelta dell'utente.

Lo sviluppatore non ha accesso a questi file in nessun momento.

👶

Minori

L'app è destinata a operatori adulti. I nomi degli alunni inseriti sono dati gestiti direttamente dall'utente sul proprio dispositivo e non vengono mai trasmessi o elaborati da terzi. Lo sviluppatore non ha accesso a tali informazioni.

What is Flory Planner

Flory Planner is a personal Android application designed for school assistants and educational support workers. It allows users to log working hours with students, view them on a monthly calendar, consult a daily agenda, and generate a monthly hours report.

The app is designed for individual use and does not require any account, registration, or online services.

✅ No data is collected

Flory Planner does not collect, transmit, or share any personal data with third parties, developers, or external servers.

  • No internet connection required
  • No account or registration needed
  • No data sent to external servers
  • No analytics, tracking, or advertising
  • No sensitive permissions requested
💾

How data is stored

All data entered (student names, schools, sessions, times) is saved exclusively on the user's device, in the app's private storage using the localStorage technology of the built-in browser (WebView).

Data never leaves the device unless the user manually exports a backup file through the app's "Export" feature.

📤

Export / Import Feature

The app allows exporting data as a JSON file and importing it later. This file is created on the device and shared through Android's sharing system (e.g., email, WhatsApp, Google Drive) entirely at the user's discretion.

The developer has no access to these files at any time.

👶

Minors

The app is intended for adult operators. Student names entered are data managed directly by the user on their own device and are never transmitted or processed by third parties. The developer has no access to such information.
